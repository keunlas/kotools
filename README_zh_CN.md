# Introduction

KoTools 这个项目包含了我在闲的无聊时写的代码。涉及自造的轮子、对开源库 C API 的封装，还有一些工作中实用的小工具。

整个项目使用 CMake 作为构建工具，使用 C++20 标准书写代码，对开源库的 C API 采用 C99 标准构建。
